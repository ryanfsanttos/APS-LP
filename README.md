## 🔐 Quiz de Segurança Digital — Sistema de Provas em Java

Projeto desenvolvido como APS (Atividade Prática Supervisionada) para a disciplina de Lógica de Programação, com o objetivo de aplicar os conceitos fundamentais da linguagem Java em um sistema interativo e educativo.
__________________________________________________________________________________________________________
## 📋 Sobre o Projeto
Este sistema é um quiz interativo sobre Segurança Digital, desenvolvido inteiramente em Java. O usuário navega por um menu principal, realiza provas sobre temas relevantes de segurança na internet e, ao final, recebe uma avaliação com base na sua média geral.
O projeto foi construído com foco em clareza de código, organização lógica e boas práticas de programação.
__________________________________________________________________________________________________________
## 🎯 Funcionalidades

✅ Menu interativo com navegação por opções numéricas
✅ 3 provas independentes com 4 perguntas cada
✅ Verificação de respostas com feedback imediato
✅ Cálculo automático da nota por prova
✅ Controle para impedir que a mesma prova seja feita duas vezes
✅ Cálculo da média geral apenas das provas realizadas
✅ Resultado final com classificação por categoria
✅ Indicação visual das provas já concluídas no menu
__________________________________________________________________________________________________________

## 🧠 Conceitos Aplicados
Este projeto foi uma oportunidade de colocar em prática de forma integrada:

Escopo de variáveis — entender onde cada variável vive e por quê.
Flags booleanas — controlar estados do programa de forma limpa e realizar verificações exatas.
Estruturas de repetição — manter o menu ativo até o usuário sair.
Estruturas de decisão — tratar cada opção e calcular resultados.
Concatenação de strings — exibir mensagens dinâmicas ao usuário.

